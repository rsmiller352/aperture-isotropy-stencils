# Aperture–Isotropy Stencils

This repository contains numerical diagnostics and reference implementations
supporting the paper:

**Unifying Aperture: A Relational Foundation for Information, Quantization, Mass,
and Spectral Staircases**  
Ray–Scott Miller

## Contents
- Discrete stencil operators (SC and FCC)
- Anisotropy metrics and isotropic horizon computation
- Scripts reproducing figures in the manuscript

## Requirements
- Python ≥ 3.9
- numpy
- matplotlib

## Usage
```bash
python isotropy_theorem.py
python isotropy_3d_lensed.py
python discrete_equilibrium_gap.py
